# Summary

One day, the button beside Pebble 2 got torn.
I made these parts for repair.
It must be printed with __flexible filaments.__

After all, I printed a mold and made a silicon button with a mold.
When printing a mold, it is better to tilt it by 45 degrees as shown in the attached image.

The original data was uploaded from [Pebble's GitHub page](https://github.com/pebble/pebble-3d) and was accompanied by this message:
